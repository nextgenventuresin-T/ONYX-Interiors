# Drop your media here

The site reads images and video from this folder. Until a file exists, you'll
see an elegant gold gradient placeholder labelled with the subject — so the
layout always looks finished. Save your downloaded Pinterest images/clips to
these exact paths and they appear automatically (no code changes needed).

## Video

| Path | Used for |
| --- | --- |
| `videos/hero.mp4` | Full-screen hero background clip |

> Keep the hero clip muted-friendly and ~10–20s, H.264 .mp4, ideally < 8 MB.
